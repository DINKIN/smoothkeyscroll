Smooth Key Scroll

Chrome Extension

Jerky scroll? No more.

Ignoring keyboard repeat rate, Smooth Key Scroll provides a smooth and responsive scrolling experience.

Arrow keys to move arround, Control (ctrl) and Option (alt) to increase/decrease speed.


View on Chrome Web Store

GPU acceleration

Although not mandatory, enabling GPU compositing and disabling VSync, can improve smoothness drastically.*

If you choose to do so, open a new tab and type: chrome://flags
After pressing return, you should be able to find the following two options, which after being enabled, should like this:

For the changes to take effect, restart Google Chrome.

*Note: Chrome flags should be modified with caution. Although no issues have been found so far, some people might experience hardware incompatibilities. If undesirable side effects are noticed, you should probably revert the settings back to original.
